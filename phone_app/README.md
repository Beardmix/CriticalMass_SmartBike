Compile and run the project for Android on Windows:
0. > cmd 
1. > cd phone_app
2. > npm install
3. > npm run build
4. > npx cap add android
5. > xcopy /S /Y .\android_res\* .\android\
6. > ionic capacitor run android
7. Android Studio > run app


Dependencies verification: npm-check -u

Capactitor updates:
> ionic cap doctor
